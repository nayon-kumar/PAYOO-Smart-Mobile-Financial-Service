## Overview

**Payoo** is a simple **digital payment simulation web application** built using **HTML, Tailwind CSS, DaisyUI, and JavaScript**.  
It allows users to perform common mobile banking operations like **Add Money, Cashout, Transfer Money, Pay Bills, and View Transaction History**.

## 🛠 Technologies Used

- **HTML5** – Structure of the application
- **Tailwind CSS** – Utility-first styling
- **DaisyUI** – UI components
- **JavaScript (Vanilla JS)** – Application logic
- **Font Awesome** – Icons
- **Google Fonts (Outfit)** – Typography

## 🚀 Features

- User login system
- Available balance display
- Add money from bank accounts
- Cashout to agent number
- Transfer money to another user
- Pay different types of bills
- Bonus coupon system
- Transaction history tracking
- Interactive alert modal system
- Responsive mobile-first UI
